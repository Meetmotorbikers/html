# Meetmotorbikers - HTML Layout
**Build with Bootstrap v4.0**


**How to install?**
_cd root/assets_
_npm install_ 
_*change css?* -> cd root/assets/ "grunt watch" -> cd stylesheets/* change files.._
_*change js?* -> cd root/assets/ "grunt watch" -> cd javascripts/* change files.._
